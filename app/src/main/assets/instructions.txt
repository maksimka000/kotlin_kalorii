Инструкция по использованию приложения "Подсчёт калорий"

1. Главный экран
- На главном экране отображается ваша дневная норма калорий
- Показывается прогресс потребления калорий за день
- Отображается количество оставшихся калорий
- Доступны кнопки для перехода в основные разделы приложения

2. Дневник питания
- Просмотр всех добавленных продуктов и рецептов
- Добавление новых продуктов через кнопку "Добавить продукт"
- Добавление рецептов через кнопку "Добавить рецепт"
- Каждая запись содержит название, количество калорий и время добавления
- Для рецептов дополнительно отображаются ингредиенты

3. Добавление продукта
- Введите название продукта
- Укажите количество калорий
- Нажмите кнопку "Сохранить"
- Продукт будет добавлен в дневник питания

4. Рецепты
- Создание новых рецептов
- Указание названия, ингредиентов и калорийности
- Возможность добавления рецепта в дневник питания
- Просмотр всех созданных рецептов

5. Цели и прогресс
- Установка дневной нормы калорий
- Отслеживание прогресса по дням
- Просмотр статистики потребления калорий

6. Настройки
- Изменение цвета фона приложения
- Настройка уведомлений
- Сброс данных приложения
- Другие параметры приложения

7. Советы по использованию
- Регулярно обновляйте дневник питания
- Следите за дневной нормой калорий
- Используйте рецепты для планирования питания
- Настраивайте приложение под свои предпочтения

8. Важная информация
- Все данные сохраняются автоматически
- Приложение работает в офлайн режиме
- Для корректной работы рекомендуется регулярно обновлять данные
- При возникновении проблем попробуйте перезапустить приложение
